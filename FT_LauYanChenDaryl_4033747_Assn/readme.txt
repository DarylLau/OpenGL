controls 
============
Press 'ESC' to quit

Press 'n' to toggle 'night' vs 'day' illumination settings

Press 'h' for Help display		

Press 'r' to reset camera		

Press 'b' to cycle thru the following states:

state 1 : Wireframe mode with Black background
state 2 : Wireframe mode with White background
state 3 : Wireframe mode with Blue  background
state 4 : Solid     mode with Primitives coloured w/o textures or lighting
state 5 : Solid     mode with Primitives shaded using lighting
state 6 : Solid     mode with Primitives textured w/o lighting
state 7 : Solid     mode with Primitives shaded using lighting and textured

'Up'    arrow key to move   forward
'Down'  arrow key to move   backward
'Left'  arrow key to strafe leftward
'Right' arrow key to strafe rightward

'Pg Up'   to float vertically up
'Pg Down' to float vertically down

'w' key to rotate camera up    (pitch up)
's' key to rotate camera down  (pitch down)
'a' key to rotate camera left  (yaw left)
'd' key to rotate camera right (yaw right)
'q' key to roll   camera left  (roll left)
'e' key to roll   camera right (roll right)

key 1 - 5  to switch on and off individual light

'f' key to toggle flashlight on/off
'i' key to toggle flashlight intensity (low, medium, high)
'c' key to toggle flashlight color (red, green, blue, white)



'p' to take picture
'm' to toggle motion blurring
'x' to toggle full scene anti-aliasing
